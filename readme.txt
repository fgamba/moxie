== Description ==

This plugin allows to show movies using a shortcode [list-movies] by creating a custom post type (movie) and provide a list of the created posts using JSON format.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create the movies using the new post type (movie)
